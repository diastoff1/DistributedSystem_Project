Nesse projeto, foi criada uma implementação de sistema distribuído utilizando somente a biblioteca sockets, em python.

Existem 3 servidores, 1 cliente, e 1 gerenciador. É necessário que execute primeiro os servidores e o gerenciador, e depois executar o cliente, que funciona como interface.

É possível fazer upload de arquivo nos servidores, os quais são replicados como forma de backup.

Esse projeto foi um projeto final da disciplina Sistemas Distribuídos.
